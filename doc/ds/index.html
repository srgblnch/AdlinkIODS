Introduction

This device server implements support for several Adlink boards with analog input/output capabilities. Supported devices are:

  • Adlink D2K devices: DAQ_2205, DAQ_2213, DAQ_2502, DAQ_2005, DAQ_2010
  • Adlink PSDASK devices: PCI_6202, PCI_6208V, PCI_6208A, PCI_9116

Some of these boards have several different capabilities. That's why the server exports several device classes:

  • AdlinkAI
  • AdlinkAO
  • AdlinkDIO
  • AdlinkIOCounter

You can have several devices in the same server with equal BoardId and BoardType but different class, for example if it has independent analog input and output channels and a some digital ones too.

Note that the AdlinkAI and AdlinkAO classes implement an almost equal interface.

All the devices in one server must refer to the same physical board.
